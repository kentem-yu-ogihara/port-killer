# port-killer

TUIでリッスン中のポートを一覧表示し、プロセスをkillできるCLIツール。

## モチベーション

複数のアプリを並行開発していると、以前起動したサーバがポートを占有したまま残り、新しいサーバの起動が失敗したり、5174とか別のポートで起動すること、ありますよね。
そのたびに ターミナルを移動してプロセスkillしたり、`lsof -i :8080` でPIDを調べて `kill` するのが面倒で、手早く片付けられるツールが欲しくて作りました。

## インストール

```bash
cargo install --path .
```

## 使い方

### TUIモード

```bash
pk
```

起動するとリッスン中のポート一覧が表示されます。

| キー | 動作 |
|------|------|
| `j` / `↓` | 下に移動 |
| `k` / `↑` | 上に移動 |
| `x` / `Delete` | 選択中のプロセスをkill |
| `/` | 検索モードに入る |
| `r` | 手動リフレッシュ |
| `q` / `Esc` | 終了 |

**検索モード中:**

| キー | 動作 |
|------|------|
| 文字入力 | ポート番号・プロセス名・プロトコルで絞り込み |
| `Backspace` | 1文字削除 |
| `x` / `Delete` | 選択中のプロセスをkill |
| `Esc` | 検索モードを終了 |

killを実行すると確認ダイアログが表示されます。`y` で実行、`n` / `Esc` でキャンセル。

ポート一覧は5秒ごとに自動更新されます。

### CLIモード（ワンショット）

ポート番号を引数に渡すと、確認なしに即座にkillします。

```bash
pk 5173
```
